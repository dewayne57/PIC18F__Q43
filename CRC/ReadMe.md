# CRC Example Family

This family demonstrates cyclic redundancy check generation and verification workflows.

## What It Is
Cyclic Redundancy Check (CRC) is a hardware data integrity engine that computes
checksum signatures over byte streams.

## What It Does
- Calculates CRC values using selected polynomials.
- Verifies data blocks against expected signatures.
- Offloads integrity checking from software loops.

## Common Uses
- Firmware image and configuration validation.
- Communications frame integrity checks.
- NVM content verification after write operations.
