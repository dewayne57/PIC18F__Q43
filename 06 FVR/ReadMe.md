# FVR Example Family

This family demonstrates Fixed Voltage Reference bring-up and analog subsystem integration.

## What It Is
Fixed Voltage Reference (FVR) is an internal precision reference source used by
analog peripherals.

## What It Does
- Generates stable internal reference voltage levels.
- Feeds ADC, comparator, DAC, and related analog blocks.
- Improves measurement consistency versus VDD-referenced operation.

## Common Uses
- Accurate ADC conversion reference.
- Comparator threshold baselines.
- Internal analog calibration workflows.
